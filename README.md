# aws-terraform-3tier-architecture
# AWS 3-Tier Architecture using Terraform

## 🚀 Overview

This project provisions a highly available AWS infrastructure using Terraform.

## 🧰 Tech Stack

* AWS (VPC, EC2, ALB, RDS)
* Terraform

## 📌 Features

* Multi-AZ deployment
* Public & private subnets
* Load balancing
* Secure architecture

## ▶️ How to Run

terraform init
terraform plan
terraform apply
